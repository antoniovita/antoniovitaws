# Antonio Vita — Portfolio & Website
A personal portfolio and website showcasing my projects, experience, and background.

## Stack
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Swiper
- React Icons

## Pages
- `/` — Home
- `/about` — About
- `/projects` — Projects
- `/experience` — Experience

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Scripts
- `npm run dev` — Start the dev server
- `npm run build` — Build for production
- `npm run start` — Run the production server
- `npm run lint` — Lint the project
